yeoh
================

Yeoh Model: A three-term model good for modeling large deformations, based only on the first invariant.

.. math::

   W = C_1 (\bar I_1-3) + C_2 (\bar I_1-3)^2 + C_3 (\bar I_1-3)^3 + W_{\mathrm{vol}}
