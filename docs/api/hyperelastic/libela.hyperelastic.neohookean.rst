Neo-Hookean Model: A simple, isotropic, incompressible (or compressible) hyperelastic model for rubber-like materials.

.. math::

   W = \frac{\mu}{2}(I_1 - 3) \quad \text{(incompressible)}

   W = \frac{\mu}{2}(\bar I_1 - 3) + \frac{K}{2}(J-1)^2 \quad \text{(nearly-incompressible)}
